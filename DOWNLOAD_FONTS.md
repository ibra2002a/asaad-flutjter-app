# تحميل خط Cairo

قم بتحميل خط Cairo من Google Fonts وضع الملفات هنا:

## الرابط:
https://fonts.google.com/specimen/Cairo

## الملفات المطلوبة:
- Cairo-Regular.ttf    (400)
- Cairo-Medium.ttf     (500)
- Cairo-SemiBold.ttf   (600)
- Cairo-Bold.ttf       (700)
- Cairo-ExtraBold.ttf  (800)
- Cairo-Black.ttf      (900)

## أو استخدم الأمر التالي:
```bash
cd assets/fonts
curl -L "https://fonts.gstatic.com/s/cairo/v28/SLXgc1nY6HkvalIvTp0y.woff2" -o Cairo-Regular.ttf
```

## ملاحظة:
يمكنك أيضاً استخدام حزمة google_fonts في pubspec.yaml
التي تحمّل الخط تلقائياً بدون ملفات محلية.
